# User Guide

This app helps you schedule youth wrestling meets.

## Sign in
Use your username and password. If MFA is enabled, enter the 6-digit code.

## Teams
Create teams, add wrestlers, or import rosters via CSV.

## Meets
Create meets with 2-4 teams, generate pairings, assign mats, and print schedules.

### Create a meet
- Use **Create New Meet**
- Choose teams and settings
- Use **Generate Automatic Pairings** if you want the system to build bouts after attendance
- The attendance dialog always appears before auto-pairings run
 - **Girls wrestle girls** (default on) limits pairings to same-sex matchups

### Delete / restore meets
- Delete moves a meet to the recycle bin (not permanent)
- Use **Restore Deleted** on the Meets page to recover deleted meets
- **Purge** permanently deletes a meet
## Enter results

On a meet page, each bout has a **Result** section.

For each bout you can record:
- Winner (choose one of the two wrestlers)
- Result type (example: DEC, MAJ, TF, FALL)
- Score (example: 6-2)
- Period and time (optional)

Click **Save** for the bout. Results will be stored with the meet.
